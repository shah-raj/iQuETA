How iQuETA
Works?

iQuETA which stands for Intelligent Question Extractor and Test Assesment is a platform which where faculties can generate test and student can appear that test in the same platform.

Steps to use iQuETA:

  • + Upload a file.
  • + Select questions, add questions and create test.
  • + Share test code.
  • + Appear test.
